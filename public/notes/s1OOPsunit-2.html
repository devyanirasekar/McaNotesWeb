Overloading:


- Creating two or more members that have the same name but are different
in number or type of parameter is known as overloading.
- Same class name and methods names but different parameters in called as Method
overloading.
- When a single Object has multiple behaviours.
- Overloading is that in which a Single Object has a same name and Provides Many
Functions.
- There are two types of overloading in C++: Operator overloading and Function
overloading.
1)Operator Overloading
- Operator overloading is a compile-time polymorphism in which the
operator is overloaded to provide special meaning to the user-defined data type.
- To assign more than one operation on some operator known as operator
overloading.
- we can change the way operators work for user-defined types like
objects and structures. This is known as operator overloading. 
-  Operator overloading is used to overload or redefines most of the operators
available in C++.
- It is used to perform the operation on the user-defined data type.
- For example, C++ provides the ability to add the variables of the user-defined data
type that is applied to the built-in data types.
- Two type of operator overloading in C++,
a) Unary Operator Overloading
b) Binary Operator Overloading
- Operator function, you can write operator () in two way-
1)Class Function
2)Friend Function
- Operator that cannot be overloaded are as follows:
o Scope operator (::)
o Sizeof
o member selector(.)
o member pointer selector(*)
o ternary operator(?:)
Syntax of Operator Overloading
class className {
... .. ...
public
returnType operator symbol (arguments) {
... .. ...
}
... .. ...
};
Here,
 returnType is the return type of the function.
 operator is a keyword.
 symbol is the operator we want to overload. Like: +, &lt;, -, ++, etc.
 arguments is the arguments passed to the function.
 
Function Overloading
- When we have two functions with the same name, but with a different
number or/and type of arguments in the code, it is known as function
overloading in C++.
- Function overloading is a feature of object-oriented programming
where two or more functions can have the same name but different
parameters. When a function name is overloaded with different jobs it
is called Function Overloading.
- In Function Overloading “Function” name should be the same and the
arguments should be different.
- Parameters should have a different type
add(int a, int b){
}
add(double a, double b){
}
- The advantage of Function overloading is that it increases the readability of the
program because you don&#39;t need to use different names for the same action.
- // same name different arguments
- int test() { }
- int test(int a) { }
- float test(double a) { }
- int test(int a, double b) { }
// Error code
int test(int a) { }
double test(int b){ }
Causes of Function Overloading:
o Type Conversion.
o Function with default arguments.
o Function with pass by reference.
Parameters should have a different data type
add(int a, int b)
add(double a, double b)

Constructor Overloading in C++:
- Constructors can be overloaded in a similar way as function overloading.
- Constructor is a member function of a class that is used to initialize the objects of the
class.
- Constructors do not have any return type and are automatically called when the
object is created.
- Overloaded constructors have the same name of the class but the different number of
arguments.
Depending upon the number and type of arguments passed, the corresponding
constructor is called constructor overloading.
Characteristics of constructors
- The name of the constructor is the same as the class name
- No return type is there for constructors
- Called automatically when the object is created
- Always placed in the public scope of the class
- If a constructor is not created, the default constructor is created automatically
and initializes the data member as zero
- Declaration of constructor name is case-sensitive
- A constructor is not implicitly inherited
Syntax:
Class class_name
{
Class_name()
{
}
Class_name(parameter)
{
}
Class_name(Parameter1, parameter2)
{
}
}
Programs:
Example 1:
#include &lt;iostream&gt;
using namespace std;

Rect(int a, int b){

cout &lt;&lt; &quot;The area is: &quot; &lt;&lt; area &lt;&lt; endl;

C++ this Pointer
- In C++ programming, this is a keyword that refers to the current instance of
the class. There can be 3 main usage of this keyword in C++.
o It can be used to pass current object as a parameter to another method.
o It can be used to refer current class instance variable.
o It can be used to declare indexers.
- Every object in C++ has access to its own address through an important
pointer called this pointer.
- The this pointer is an implicit parameter to all member functions. (implicit –
means to reduce the length of arguments list of function class without
restoring to the variable.)
- Therefore, inside a member function, this may be used to refer to the
invoking object.
- Friend functions do not have a this pointer, because friends are not members
of a class. Only member functions have a this pointer.
- The main purpose of the “this” pointer is to be used to refer to the calling
object within a member function.
-
Example:
1) When local variable’s name is same as member’s name
#include&lt;iostream&gt;

/* local variable is same as a member&#39;s name */

Static data members
- Static is a keyword in C++ which is used to declare a special type of a variable or
a function inside or outside of a class.
- static keyword is mainly used for memory management.
- It can be used with variables, methods, blocks and nested classes.
- It is a keyword which is used to share the same variable or method of a given class.
- Basically, static is used for a constant variable or a method that is same for every
instance of a class.
- Static data members are class members that are declared using the static keyword.
- There is only one copy of the static data member in the class, even if there are many
class objects.
-  This is because all the objects share the static data member.
- The static data member is always initialized to zero when the first class object is
created.
- The syntax of the static data members is given as follows −
static data_type data_member_name;
static data_member;
static data_type identifier;
Example:
static data_member;
static data_type identifier;
For example:
class A{
static int x;
};
class A{
//Declaration
static int x;
};
//Defination
int A::x=0;
